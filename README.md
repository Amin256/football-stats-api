# Football Stats API

A RESTful API for football match statistics and analytics built using FastAPI.

## Features

- CRUD operations for football teams and matches
- Analytical endpoints for league tables and statistics
- SQL database storage
- Swagger API documentation

## Technologies

- FastAPI
- SQLAlchemy
- SQLite
- Python

## Running the API

Clone the repository:

git clone https://github.com/yourusername/football-stats-api.git

Install dependencies:

pip install -r requirements.txt

Run the server:

uvicorn app.main:app --reload

Open documentation:

http://127.0.0.1:8000/docs
